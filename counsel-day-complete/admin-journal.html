Counsel Journal · controls

Everything that drives the Journal product: which AI model writes the weekly + monthly verdicts, the Whisper variant that handles voice transcription, the subscription price, the prompt links, and the live cost meters. Counsel Journal is a paid-only subscription at $4.99 USD/month.

AI models

Live model selectors for the verdict synthesis (Claude) and the voice transcription (OpenAI Whisper). Changes save instantly and apply on the next cron run.

Weekly verdict model
Used by the Sunday-evening verdict cron. Anthropic Pricing: Opus ~$15/M input, Sonnet ~$3/M, Haiku ~$0.80/M.
Monthly verdict model
Used on the first Monday of the month for all active Counsel Journal subscribers. Reads the past 4 weekly verdicts together; Opus recommended for the synthesis quality.
Voice transcription
~$0.006 USD per minute. Audio is sent to OpenAI; the no-train data agreement applies. Audio is NOT stored after transcription (see vault settings below).
Anthropic API key
checking… Set in /etc/counsel-day-app/env.local as ANTHROPIC_API_KEY. Rotates via Infisical secrets manager.
OpenAI API key
checking… Set in /etc/counsel-day-app/env.local as OPENAI_API_KEY. Used for Whisper transcription only.
 

Subscription

Counsel Journal is a single paid subscription · no free tier. Changes to the subscription price must be made in the Stripe Dashboard so that webhook reconciliation stays in sync.

Monthly price
Stripe price ID: STRIPE_DAILY_PRO_PRICE_ID in env.local. Read-only here.
