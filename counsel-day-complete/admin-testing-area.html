Testing area · run a verdict on fixture data

Operator sandbox for prompt-tuning, tone checks, and AI-output spot-checks. Hits the real Anthropic API with the production system prompt. Nothing is persisted · no rows in decisions, votes, verdicts, or verdict_runs. Each run costs roughly $0.02-$0.05 of Anthropic credits.

How it works: Fill in the question, choose a tier, give each participant a name, then click through 7 days of votes (and notes, if you want to test tone-handling). Click Run verdict and the AI synthesis appears below alongside the numerical per-participant summary. Solo Free skips the AI call and returns numbers only · same as the customer-facing experience for that tier.
