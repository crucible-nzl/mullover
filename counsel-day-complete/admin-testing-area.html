Testing area · run a verdict on fixture data

Operator sandbox for prompt-tuning, tone checks, and AI-output spot-checks. Hits the real Anthropic API with the production system prompt. Every run is persisted to verdict_test_runs and visible under the Testing tab in verdict logs for later review. Each run costs roughly $0.02-$0.05 of Anthropic credits.

How it works: Fill in the question, choose a tier, give each participant a name, then click through 7 days of votes (and notes, if you want to test tone-handling). Click Run verdict and the AI synthesis appears below alongside the numerical per-participant summary. Solo Free skips the AI call and returns numbers only · same as the customer-facing experience for that tier.

Picking a preset auto-fills the form. The "Reset to default fixture" button regenerates the current tier's stock fixture (handy if you changed duration or wiped participants).
