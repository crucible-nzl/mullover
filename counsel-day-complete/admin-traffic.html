Traffic · GA4

Live data from Google Analytics 4 via the Data API.

GA4 not connected. Set GA4_SERVICE_ACCOUNT_JSON and GA4_PROPERTY_ID in /etc/counsel-day-app/env.local on the server, then restart counsel-day-app. Steps:
  1. Create a Google Cloud service account; grant it Viewer on this GA4 property.
  2. Download the JSON key; paste the whole file as the value of GA4_SERVICE_ACCOUNT_JSON.
  3. Set GA4_PROPERTY_ID to the numeric property id (NOT the measurement id G-XXX).
  4. Run npm install @google-analytics/data on the server (already declared in package.json).
  5. Restart: sudo systemctl restart counsel-day-app.
